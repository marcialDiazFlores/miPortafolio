Interfaz/aplicación web para hacer CRUD sobre una base de datos de productos en MongoDB. Al ingresar un producto se crea la base de datos y la colección
en mongo, y se visualizan los productos en pantalla.

Ejecutar el script app.py y ir a la dirección http://127.0.0.1:4000 en el navegador.

Hecho con python + mongoDB (pymongo), html + css + bootstrap y flask.
