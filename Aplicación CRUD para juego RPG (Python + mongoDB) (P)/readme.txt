Aplicación python con interfaz gráfica para hacer CRUD sobre una base de datos mongoDB, la cual almacena datos de un videojuego RPG.
La aplicación sigue el patrón de diseño MVC (Modelo - Vista - Controlador) y posee varias clases para cada una de las entidades del videojuego.

Para ejecutar la aplicación se debe ejecutar el script "Interfaz CRUD.py" que está dentro de la carpeta Vista.Aplicación python con interfaz gráfica para hacer CRUD sobre una base de datos mongoDB, la cual almacena datos de un videojuego RPG.
